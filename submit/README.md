# 📁 Submit Folder / 提交資料夾說明

This folder stores all **model output files** generated by the pipeline.

本資料夾將用來存放 pipeline 執行後所產生的 **模型輸出與提交檔案**。

---

## 📄 Output Files (Generated Automatically)

執行 `python main.py` 後會自動產生以下 CSV：

```
submit_stack_topk.csv        # 競賽最終提交檔
acct_predict_out_stack.csv   # 模型完整分數（debug / analysis）
```

> ⚠️ 本資料夾預設為空。
> 檔案會在 pipeline 執行時自動生成。

---

## ▶️ Notes 注意事項

* 請勿手動放入非必要檔案
* 若清空本資料夾不會影響訓練，只會刪除輸出結果
* 請確認 data/ 已放入原始資料後再執行 pipeline
